📌 Bookmark Manager
    A modern and powerful bookmark manager built with Next.js and TypeScript that helps developers organize, search, and filter their favorite tools, resources, and links efficiently.

🚀 Overview
    This project allows users to:
    - Save and manage bookmarks
    - Search bookmarks in real-time
    - Filter by tags and categories
    - Discover trending and popular resources
    - Pin important bookmarks for quick access

It is designed to feel like a real-world developer tool — fast, scalable, and intuitive.

✨ Features
    🔍 Real-time search
    🏷️ Tag & category filtering
    📌 Pin bookmarks
    📊 View tracking (popularity system)
    ⚡ Fast performance with Next.js
    🎯 Clean and responsive UI
    🧠 Smart filtering logic (combined search + tags)

🛠️ Built With
    - Next.js – React framework for production apps
    - TypeScript – Type safety and better developer experience
    - Tailwind CSS – Utility-first styling
    - React Hooks – State and logic handling

🧠 What I Learned
    While building this project, I improved my understanding of:
    - Managing derived state vs source state
    - Building scalable filtering systems
    - Combining multiple filters (search + tags + categories)
    - Structuring data for real-world applications
    - Avoiding common React pitfalls like:
    - unnecessary useEffect
    - state mutation issues
    - infinite re-renders
